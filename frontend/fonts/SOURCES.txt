Self-hosted webfonts for PocketLog.

PocketLog ships all third-party assets from its own origin (no CDNs,
no tracking — see CLAUDE.md §Drittanbieter & Privacy). This file
documents where each .woff2 here came from, so its identity and
integrity can be verified at any time.

License for all files in this directory: SIL Open Font License 1.1
(see OFL.txt next to this file for the full license text + copyright
holders for each family).

Verify locally with: sha256sum -c SOURCES.txt --ignore-missing


DM Sans (variable, weights 300–700)
-----------------------------------
- Source:        https://github.com/googlefonts/dm-fonts
                 (release: https://github.com/googlefonts/dm-fonts/releases/tag/v1.103)
- Google Fonts:  https://fonts.google.com/specimen/DM+Sans
- Subset:        Two woff2 files — `latin` covers German (incl. ä/ö/ü/ß + €);
                 `latin-ext` adds Polish/Czech/Turkish so non-German
                 transaction descriptions don't drop into the system font.
                 Subsets and unicode-range declarations mirror Google
                 Fonts' v1 API output for DM Sans (variable axis wght 300..700).

DM Serif Display (regular, weight 400)
--------------------------------------
- Source:        https://github.com/googlefonts/dm-fonts
                 (release: https://github.com/googlefonts/dm-fonts/releases/tag/v1.103)
- Google Fonts:  https://fonts.google.com/specimen/DM+Serif+Display
- Subset:        latin + latin-ext, same rationale as above.
